## Using Services

### Via CLI

```bash
heady research "What is quantum computing?"
heady "Write a Python sorting algorithm"
heady health
heady services
```

### Via MCP (in IDE)

All services are automatically available as MCP tools in your IDE. Type a request in the AI chat panel and Heady routes it to the optimal service.

### Via Chrome Extension

Click HeadyBuddy in your Chrome toolbar → type your request → get results from any Heady service.
